# 需求文档

> 版本：v1.0 (MVP)

## 1. 文档概述

**目标**：构建一个基于人格标签（PDB 模式）的社区，结合滑动匹配社交（Tinder 模式）与小游戏竞技的综合平台。

## 2. 用户角色

**核心逻辑**：以“人格/兴趣标签”为连接点，实现从“内容讨论”到“关系建立”再到“共同游玩”的闭环。

| 角色 | 描述 |
|------|------|
| 游客 | 仅能浏览社区热门内容，无法评论、匹配或进入游戏排行榜。 |
| 注册用户 | 拥有唯一账户，可参与社区讨论、左右滑动匹配、游玩游戏并记录成绩。 |
| 管理员 | 负责后台内容审核、用户封禁、游戏分数清零。 |

## 3. 功能模块详细说明

### 3.1 账户中心 (Identity Server)

- **注册登录**：仅支持邮箱。注册后发送激活邮件（后期校验）。
- **多应用属性**：一个账户关联多个 Profile。
- **社区 Profile**：MBTI 类型、兴趣标签、个人简介。
- **游戏 Profile**：游戏等级、最高分记录、排行榜昵称。

### 3.2 社区讨论 (The "PDB" Module)

- **板块分类**：按人格类型（如 INTP, ENTJ）或兴趣主题划分。
- **信息流**：支持发布图文。排序逻辑：`Score = (点赞数 + 评论数 * 2) / (小时数 + 2)^1.5`（基础衰减算法）。
- **互动**：点赞、二级评论。

### 3.3 匹配社交 (The "Match" Module)

- **滑动卡片**：展示其他用户的标签和照片。
- **匹配逻辑**：
  - 右滑（Like），左滑（Pass）。
  - **双向匹配 (Match)**：当双方均 Like 时，系统自动建立私聊会话。
- **推荐算法**：优先推荐标签重合度高或活跃度高的用户。

### 3.4 游戏中心 (The "Arcade" Module)

- **小游戏**：内置 H5 游戏（如跳一跳、合成大西瓜等）。
- **全球排名**：实时更新的 Top 100 排行榜。
